Hi Bruce,

in fact one has to be cautious as to "false positives" when using this method, especially for 1-syllable words like "(h)ear, "air" and "I". Some of the SpeakJet allophones are diphthongs, e.g. #22 ("IYRR" like in "Clear, Hear, Year"), #23 ("EYRR" like in "Hair, Stair, Repair"), or #30 ("OHIH" like in "Sky, Five, I"). There are only 72 different allophones, so the probability that one-syllable words like "(h)ear/here" , "air", and "I" appear, is relatively high.

I don't assume that everything that is received by this method is true communication. As with every method, it depends on various factors like meaningfulness, relatedness, length of the message, etc., if a message is regarded as a valid EVP.

Stefan
