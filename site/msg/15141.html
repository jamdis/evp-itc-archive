An op amp is an "operational amplifier" which is essentially treated as a
little box that amplifies linearly whatever is put into it. By using
different resistors in a circuit you can adjust the gain of the op amp
circuit.

I'm not sure exactly how Frank is using one for the tuning. It's not very
common to use an op amp as a noise source but I think it's been done. There
are better noise sources out there.
