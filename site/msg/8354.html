Hi Kristin,

start EVPmaker, and open the WAV file in it. You can do this by using
the Open button, or simply by "drag'n'drop" the file into EVPmaker. Next
check "Use Cue List", and click on the button with the yellow arrow to
start playback. You can record the sound as usual to any
cassette/digital recorder or even into a recording software on the same
computer.

In my short test last night, I used the "EVP Session" function, which
however didn't record my own voice - but it was the easiest way to test
it. Using "EVP Session" is a bit complicated to explain, but it's all
written in the manual... ;-)

Hope this helps...
