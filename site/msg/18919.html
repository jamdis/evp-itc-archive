A friend of mine did a series of experiments using a faraday cage with evp. A radio was connected via patch cord to a cassette recorder. The radio was de-tuned so was only picking up static. The recorder was switched on and the 2 units were placed inside a 44 gallon drum and the lid sealed on (excellent faraday cage). Result - evp voices recorded.
The conclusion reached is that evp is neither a radio signal nor accoustic, as there was no microphone and no stations able to be received. It is likely therefore that evp affects the electronics inside the radio and tape recorder BUT a carrier signal of some sort is always needed - like the internally generated white noise the radio was producing inside the drum - and the white noise inside the recorder.

Radio sweep wouldnt work inside a (good) faraday cage. It requires radio station segments to be efficient, but as mentioned it would however 'fall back' to conventional evp mode if starved of voice material, and produce some white noise based evps.
JEFF

An optimum faraday cage would need to be ferrous based metal, so it can also shield magnetic components of EM fields. Aluminum foil doesnt do this. Id recommend an ammo box.
JEFF
