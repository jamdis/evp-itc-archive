Hi Misty, "inner energy", all,

maybe one could feed the SpeakJet allophones into EVPmaker. The attached
WAV file contains the 72 allophones of the SpeakJet chip. (I used
Audition to cut them out of the sample clip which has been posted here
recently, and store them in a cue list.) To use this file as an EVPmaker
source, the "Use Cue List" option must be checked. I have done a few short
sessions, and got this strange message:

                "I little Sir help you eclipse."

(Doesn't make any sense to me... lol)

Of course, this cannot replace a hardware version of SpeakJet, since
EVPmaker isn't able to produce the proper transitions between the
allophones. But maybe it's worth a try...

Stefan
