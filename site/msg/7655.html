> Rhonda W <rhondak1973@...> wrote:            Hey Mike,
>   If you want to do a quick edit...you can do it using the sound 
recorder that comes with Windows. Just go to the start menu, all 
programs, accessories, entertainment, sound recorder....you can set 
the position of the marker to before the question and answer and 
delete the first part and then put the marker so its after and delete 
the rest...just make sure to save the new one u make separately from 
the .wav file otherwise you'll lose the whole thing.  I would make a 
copy of the .wav and save it else where just to be safe.  That's an 
option if ya don't know which program you want to D/L yet.  You can 
do some pretty cool stuff using that recorder too, speed things up, 
slow em down, play em backwards, add echo, etc.  Pretty nifty for 
just a basic lil thing.
