Commercial text-to-speech applications usually aren't "low-level" enough to have access to the speech elements, but perhaps research sites or university web sites are a resource for such samples?

BTW, there's a program called TalkAny from Microsoft; you can type in any text and it will speak it immediately. It has many options to alter the voice, and there's also a preset called 'Little Robot' which makes the voice sound very similar to the SpeakJet chip. I use it sometimes to counter-check / verify "EVPmaker + SpeakJet" EVPs.

Perhaps it can be used to generate an own set of phonemes / allophones.

I could upload this program, if desired.
