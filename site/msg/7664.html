Hi Mike.  You have a couple options.  You can use this free converter program (I've not 
tried it but have read from others that it's good):

http://www.softwarepile.com/audio_converter.html

Or you can play the file into Audacity via a patch cord from the ear jack of your recorder to 
your sound-in jack on your computer (rather than transferring the file using the USB).  If 
you want to export a file from Audacity as an MP3, you will also need to download and 
install the LAME add-on.  

http://audacity.sourceforge.net/download/

Hope this helps some.
