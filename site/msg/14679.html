Yes, I'd guess that some electrical circuit or device failure could cause
the tapping and EMF spikes, including a bad power meter. Usually these
things occur when something is drawing power. Bad wiring or a bad power
meter will not show up if everything in the house is turned off or
unplugged. Note some equipment like stereos, some microwave ovens, etc. have
an active part of the circuit even when when they seem to be off. A bad
transformer in a stereo could cause the problem too. When I worked in an
electronics repair shop, we had a lot of those come in for repair.

When a group I'm in does home investigations, we try to spend part of the
time with the power completely shut off. That way we can be reasonably sure
that any EMF spikes on the meter or strange popping sounds etc. aren't
coming from the house wiring or equipment. It would probably be useless if
not impossible to try this in an apartment if it is surrounded by other
apartments.
If the problem is persistent you can try to track it down by unplugging
everything in the house one at a time and see if it goes away when a
particular thing is unplugged.
