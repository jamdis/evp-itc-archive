I have sony acid pro 6, I use it for loop creation for a website Im working on currently and for a track for a animated video Im doing for the intro to my site (that Im doing in 3d studio max 9). I dont believe the new company that took it over calls it cool edit pro any longer but, I use the older version because it works great for evp's, so if you google it you can find it no problem, it makes it easier to strip out ambient noise (like white noise) while leaving the voices completely entact. its basically like diving deeper and deeper into the sound and it works great for it.
