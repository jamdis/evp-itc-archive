Hi Kerry.  Welcome!  Frank doesn't sell his boxes, but he has posted in the forum, some 
schematics if you would want to take on building one for yourself.  

Also, there are commercially available alternatives.  There is the MiniBox from 
www.paranormalsystems.com (priced $399-$449 depending if you want a crystal added to 
it or not).  Sometime in the near future, they will be launching the MaxiBox (a more 
advanced version of the MiniBox - which will remain in production for the average user.)

As well, the following Radio Shack models can be modified for continuous sweep by 
snipped the wire to the mute pin:

12-469 new or old version (new one has a click-click-clicking during sweep)
12-470 old version only (hard to come by)
12-820
20-125

And last, but certainly not least is, manually sweeping the radio (just move the tuner back 
and forth quickly so that only snippets are heard from each broadcasting station.)

Hope this info helps some.
