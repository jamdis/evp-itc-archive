Hi Steve,

the wav file has a cue list embedded into it, consisting of 72 cues - one for each allophone. If EVPmaker is set up to use the embedded cue list, the other settings such as Overlap, Zerocross, Crossfade, Segment size etc. don't apply, thus they will be greyed out. These options, and the "Use Cue List" option, exclude each other. If "Use Cue List" should be greyed out, then uncheck the other options.

Stefan
