Hi Leah.  Some months ago, Stefan created an allophones files (I think it's 72 speech 
fragments based on the Speak Jet chip) to use in conjunction with his EVP Maker program.  
Because the allophones file contains only fragments and not complete words or sentences 
(though to be sure, some fragments can sound like words, ie:  "I" "ear/hear" etc) there isn't 
an easy explanation for the responses which are produced by it ... responses that are not 
only complete sentences, but relevant to the topic of the session.

I think using the allophones file goes a long way in helping to rule out false positives.  I 
have actually had multiple sessions where real-time communication has taken place back 
and forth between myself and the person in spirit communicating via this set-up.  :-)

convo_with_arthur.mp3
