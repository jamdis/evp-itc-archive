I agree with the the thought behind the set time. A few years back I
set up an experiment with about 10 people. Some did set time and set
place. some set time, different place, some completely random times
and places, etc. The results were inconclusive and the dedication from
the experimenters was not as I had hoped it would be. 

What I would like to see you add would be a timer. Is there a
consistent time between words, responses, etc.?

My whole thought process is in developing a way or set of tests,
protocol, etc to eliminate as many other possible causes as we can. I
know we can't eliminate everything due to the simple fact that we are
using radio signals but, if we can attempt to keep it as natural as
possible and try to introduce some scientific methods. 

I have listened to many of your files and do find them compelling.
