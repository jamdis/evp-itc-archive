BTW, there's a cool function in Audition / Cool Edit, called "Dynamic
Range Processing". A quote from the Help file: "The dynamics processor
varies the output level of a waveform, based on its input level. This
lets you limit or compress the dynamic range of a sample so that the
perceived loudness is kept below a defined limit, or so that the
waveformï¿œs overall dynamic range is kept at roughly the same level."

There are some useful presets, e.g. "limit soft -18 dB w/boost" which
limits the level to -18 dB and then amplifies the signal, resulting in a
waveform with an overall equal volume.
