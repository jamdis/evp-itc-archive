Thanks! Well, the Sony Arm Band radio is almost usable, FM being 
pretty weird. AM works with the usual 8 volt tuning voltage, but FM 
is tuned from 8 volts to 13 volts, probably because FM is also used 
for TV and weather. I'll probably use it as an AM only box. 

 The Honda radio seems usable. It was new enough that it had the anti 
theft feature, requiring an access code to even turn on the radio. 
Fortunately, someone had written the code on the bottom of the radio. 
I was able to work on the pin out of the tuner modules, not only 
that, I had one of these exact same radios back in 2004, but no code, 
so never could figure out the tuners. I did keep the tuner modules, 
so hopefully I'll have two sets.

 Producing this stuff always seems to cause problems, and I don't
