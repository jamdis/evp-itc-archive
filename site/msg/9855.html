Hi Kay.  I see that it's a bug (just googled the error message).  Did you download the 
STABLE version of Audacity or the BETA?  It seems to be a problem with version 1.3.4, and 
people are saying uninstall it and use an older version.  Looks like the latest stable version 
for Windows is 1.2.6

Be sure you also get the LAME add-on which will allow you to export your files as MP3.  
Without it,  your only option is WAV, which is much larger.  

Hope this helps some.  :-)
